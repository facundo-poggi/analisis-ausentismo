# Portfolio – Análisis de Ausentismo en Salud

Este repositorio contiene proyectos desarrollados en Python enfocados en el análisis de ausentismo y gestión de turnos en un contexto de atención en salud.

## Proyectos

### Proyecto 1 – Análisis de Ausentismo
Análisis general de turnos, asistencias y ausencias a partir de un archivo CSV.
Incluye métricas básicas y un reporte de resultados.

Tecnologías:
- Python
- Manejo de archivos CSV
- Estructuras de datos (listas, diccionarios)
- Lógica de negocio

### Proyecto 2A – Ausentismo por Día
Análisis del ausentismo segmentado por día, orientado a detectar patrones temporales.

Tecnologías:
- Python
- Procesamiento de datos
- Análisis exploratorio básico

## Contexto
Los proyectos forman parte de un proceso de formación en análisis de datos, con foco en aplicaciones reales dentro del ámbito de la salud.
