Descripción

Este proyecto analiza el ausentismo en turnos médicos a partir de un archivo CSV.
El objetivo es obtener un panel general con información básica sobre los turnos, como cantidad total, ausencias y primeros registros, utilizando Python puro, sin librerías externas como pandas.

El foco del proyecto es consolidar:

lectura de archivos CSV

uso de estructuras básicas de Python

recorrido de datos con bucles

organización del código en scripts simples

Funcionalidades del panel

El script permite:

Leer el archivo CSV completo

Contar el total de turnos

Identificar turnos ausentes

Mostrar información básica de los primeros registros

Tener una visión general del ausentismo sin análisis por día

Este proyecto funciona como base lógica para análisis posteriores más específicos.

Tecnologías utilizadas

Python 3

Módulo estándar csv

Estructuras básicas (listas, diccionarios)

Objetivo educativo

Este proyecto forma parte de un portfolio de aprendizaje, con foco en:

comprender la lectura y manipulación de datos desde CSV

afianzar lógica básica sin depender de librerías externas

construir una base sólida para análisis más avanzados

Relación con proyectos posteriores

Este proyecto sirve como punto de partida para análisis más específicos, como:

ausentismo por día

métricas por profesional

análisis temporal
